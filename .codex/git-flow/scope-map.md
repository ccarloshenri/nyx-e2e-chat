# Scope Map

| Area changed | Scope |
| --- | --- |
| controllers | `Controller` |
| business rules / use cases | `Bo` |
| persistence / DynamoDB / repositories | `Dao` |
| domain entities / DTOs / enums | `Model` |
| request schemas / validation | `Validator` |
| Lambda entrypoints | `Handler` |
| decorators / wrappers | `Decorator` |
| technical services | `Service` |
| contracts / abstractions | `Interface` |
| external adapters / AWS clients | `Gateway` |
| environment / settings | `Config` |
| root or backend docs | `Readme` |
| frontend pages / components / routing | `Frontend` |
| backend-wide changes | `Backend` |
| login / JWT / session | `Auth` |
| messaging flow | `Message` |
| conversations flow | `Conversation` |
| websocket connections | `Connection` |
| client-side encryption | `Crypto` |
| unit tests | `Unit` |
| integration tests | `Integration` |
| deploy / aws / ci / repo ops | `Infra` |
